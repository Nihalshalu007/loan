KSBCDC Internal Audit System is a real-time loan document verification tool built for the Kerala State Backward Classes Development Corporation audit team. Team members can verify loans, track ongoing verifications, collaborate via chat, and review daily/historical stats — all in one place.

Core Verification
✅
Submit Verified Loans
Record verified loan documents with scheme, customer details, amount and remarks in seconds.
⏳
Ongoing / In-Progress
Mark loans as ongoing to claim them. Other team members will see a lock. Form data persists so you can continue anytime.
✏️
Edit Verified Records
Edit any verified loan unlimited times. Move a verified loan back to ongoing directly from the edit panel.
📜
Full Edit History
Every field-level change is recorded — who edited, what changed, and when — per loan.
Analytics & Stats
📊
Daily History Panel
See verified count per day in grid or bar-chart view. Days with 0 count are auto-detected as holidays.
📅
Smart Comparison
Yesterday's count stat skips holidays and compares today against the last actual working day automatically.
